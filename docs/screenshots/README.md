# Screenshots

The project README and the Joomla Extensions Directory (JED) listing reference the PNG files
below. Capture them from the component's admin views (1280×800 recommended) and drop them here
with these exact names:

| File | View | URL (admin) |
| --- | --- | --- |
| `dashboard.png` | Dashboard | `index.php?option=com_jsecdash&view=dashboard` |
| `waf.png` | Web Application Firewall | `index.php?option=com_jsecdash&view=waf` |
| `healthcheck.png` | Health Check | `index.php?option=com_jsecdash&view=healthcheck` |
| `blocks.png` | IP Blocks | `index.php?option=com_jsecdash&view=blocks` |
| `scanner.png` | File Scanner | `index.php?option=com_jsecdash&view=scanner` |

Tip: use a clean admin template (Atum), hide unrelated modules, and crop to the content area.
